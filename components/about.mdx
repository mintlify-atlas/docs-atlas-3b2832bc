# About Component

The About component provides an overview of the DEV WEEK event with a brief description and four key feature cards explaining what attendees can expect. It's the first content section after the hero.

## Overview

The About component displays:
- **Event description** - Brief explanation of DEV WEEK 2025
- **Feature cards** - Four main event highlights with icons
- **Staggered animations** - Cards animate in sequence on scroll
- **Institution reference** - Highlights the organizing program (Desarrollo de Sistemas de Información)

## Usage

The component is imported and rendered in `App.tsx` with lazy loading:

```tsx
// src/App.tsx
const About = lazy(() => import("./components/About"));

// Inside main content
<Suspense fallback={<div style={{ minHeight: "200px" }} />}>
  <About />
</Suspense>
```

## Props

The About component does not accept any props. It's a self-contained section component with hardcoded content.

### 3. Scroll-Triggered Animation

The component uses Intersection Observer with anime.js for entrance animations:

```tsx
// src/components/About.tsx:37-56
useEffect(() => {
  const observer = new IntersectionObserver(
    (entries) => {
      entries.forEach((entry) => {
        if (entry.isIntersecting && !hasAnimated) {
          anime({
            targets: `.${styles.aboutCard}`,
            translateY: [50, 0],
            opacity: [0, 1],
            duration: 800,
            delay: anime.stagger(150),
            easing: "easeOutExpo",
          });
          setHasAnimated(true);
          observer.disconnect(); // Performance: Disconnect after animating
        }
      });
    },
    { threshold: 0.2 }
  );

  if (sectionRef.current) {
    observer.observe(sectionRef.current);
  }

  return () => observer.disconnect();
}, [hasAnimated]);
```

**Animation properties:**
- `translateY: [50, 0]` - Cards slide up from 50px below
- `opacity: [0, 1]` - Fade in effect
- `duration: 800` - 800ms animation
- `delay: anime.stagger(150)` - Each card delayed by 150ms
- `easing: "easeOutExpo"` - Smooth exponential easing
- `threshold: 0.2` - Triggers when 20% of section is visible

### 4. One-Time Animation

The component uses state to prevent re-animation on scroll:

```tsx
const [hasAnimated, setHasAnimated] = useState(false);

// Animation only runs if !hasAnimated
if (entry.isIntersecting && !hasAnimated) {
  // ... animate
  setHasAnimated(true);
  observer.disconnect();
}
```

## Customization

To customize the About section:

1. **Edit event description** - Update the text in the aboutDescription paragraph (line 74-83)
2. **Modify features** - Edit the features array (lines 6-31):
   - Change icons by importing different Lucide icons
   - Update titles and descriptions
   - Add or remove features (component will adapt grid layout)
3. **Adjust animations** - Modify anime.js parameters in the useEffect (lines 42-49):
   - Change `duration` for faster/slower animation
   - Adjust `delay: anime.stagger()` for card timing
   - Modify `translateY` values for different entrance effect
4. **Update section ID** - Change `id="about"` if linking from other components

## Performance Notes

- **Lazy loaded** - Component is loaded on-demand via React.lazy()

- **Threshold optimization** - Only triggers when 20% visible (not too early)
