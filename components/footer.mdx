# Footer Component

The Footer component provides site-wide navigation, social media links, event information, and developer credits. It serves as the closing section of the DEV WEEK 2025 website.

## Overview

The Footer component provides:
- **Event description** - Brief overview of DEV WEEK 2025
- **Social media links** - Facebook, Instagram, and TikTok with icons
- **Quick navigation** - Links to main page sections with smooth scrolling
- **Copyright notice** - Institutional copyright information
- **Developer credits** - Attribution to Nyuro Frame

## Props

The Footer component does not accept any props. It's a self-contained section component.

### 3. Quick Links Navigation

Second column provides navigation to main sections:


## Accessibility Features

- **Semantic HTML**: Uses `<footer>` element
- **ARIA labels**: Social links have descriptive `aria-label` attributes
- **Link labels**: All links have clear, descriptive text
- **External links**: `rel="noopener noreferrer"` for security
- **Keyboard navigation**: All links are keyboard accessible
- **List structure**: Navigation uses proper `<ul>` and `<li>` elements

## SEO Considerations

- **Structured data**: Footer provides site-wide links for crawlers
- **Social links**: Direct links to official social media profiles
- **Copyright**: Clear ownership and date information
- **Internal links**: Helps with site structure and crawlability

## Customization

### Adding Social Links

Add new social platform links:

```tsx
<a
  href="https://linkedin.com/..."
  target="_blank"
  rel="noopener noreferrer"
  className={styles.socialLink}
  aria-label="LinkedIn"
>
  <Linkedin size={20} />
</a>
```

### Adding Navigation Links

Add more quick links:

```tsx
<li>
  <a
    href="#new-section"
    onClick={(e) => {
      e.preventDefault();
      document
        .getElementById("new-section")
        ?.scrollIntoView({ behavior: "smooth" });
    }}
    className={styles.link}
  >
    New Section
  </a>
</li>
```

### Updating Copyright Year

Update the year in the copyright text:

```tsx
<p className={styles.copyright}>
  © 2026 IESP Túpac Amaru...
</p>
```
