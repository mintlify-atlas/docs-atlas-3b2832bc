# Schedule Component

The Schedule component displays the three-day event schedule with interactive cards for each day. Clicking a day opens a modal with detailed activities and timing information.

## Overview

The Schedule component provides:
- **Day cards** - Visual cards for each event day with theme and basic info
- **Interactive modal** - Detailed schedule view for selected day
- **Download option** - Link to download the complete schedule
- **Animations** - Staggered entrance effects using Intersection Observer

## Props

The Schedule component does not accept any props. It's a self-contained section component.

  </ParamField>

  </ParamField>

  </ParamField>
  
  <ParamField path="actividades" type="Actividad[]">

  </ParamField>
  
  <ParamField path="destacado" type="boolean">

  </ParamField>
</ParamField>

## Key Features

### 1. Day Cards Grid

Displays all event days in a responsive grid:

```tsx
<div className={styles.grid}>
  {diasEvento.map((dia) => (
    <div
      key={dia.id}
      onClick={() => setSelectedDay(dia)}
      className={styles.scheduleCard}
    >
      <div className={styles.dayName}>{dia.nombre.toUpperCase()}</div>
      <h3 className={styles.cardTitle}>{dia.tema}</h3>
      <div className={styles.cardInfo}>
        <Calendar className={styles.cardIcon} />
        <span>{dia.fecha}</span>
      </div>
      <div className={styles.cardInfo}>
        <Clock className={styles.cardIcon} />
        <span>{dia.hora}</span>
      </div>
      <button className={styles.detailsBtn}>
        Ver detalles
        <ChevronRight size={16} />
      </button>
    </div>
  ))}
</div>
```

### 2. Intersection Observer Animation

Cards animate into view when scrolled into viewport:

```typescript
useEffect(() => {
  const observer = new IntersectionObserver(
    (entries) => {
      entries.forEach((entry) => {
        if (entry.isIntersecting) {
          const cards = document.querySelectorAll(`.${styles.scheduleCard}`);
          enterStagger(cards, 100);
        }
      });
    },
    { threshold: 0.1 }
  );
  if (sectionRef.current) {
    observer.observe(sectionRef.current);
  }
  return () => observer.disconnect();
}, []);
```

### 3. Modal with Activity Details

When a day is clicked, a modal displays the full schedule:

```tsx
{selectedDay && (
  <div
    className={styles.modalOverlay}
    onClick={() => setSelectedDay(null)}
  >
    <div
      className={styles.modalContent}
      onClick={(e) => e.stopPropagation()}
    >
      <div className={styles.modalHeader}>
        <div className={styles.dayModal}>
          {selectedDay.nombre.toUpperCase()}
        </div>
        <h3 className={styles.modalTitle}>{selectedDay.tema}</h3>
        {/* Date and time info */}
      </div>
      
      <div className={styles.modalBody}>
        <p className={styles.modalDesc}>{selectedDay.descripcion}</p>
        <h4 className={styles.modalSubtitle}>Actividades</h4>
        <ul className={styles.modalList}>
          {selectedDay.actividades.map((actividad, index) => (
            <li key={index} className={styles.modalListItem}>
              <div className={styles.modalListNum}>{index + 1}</div>
              <div className={styles.actividadBox}>
                {actividad.hora && (
                  <span className={styles.actividadHora}>
                    {actividad.hora}
                  </span>
                )}
                <div className={styles.actividadTituloWrapper}>
                  <span className={styles.actividadTitulo}>
                    {actividad.titulo}
                  </span>
                  {actividad.subtitulo && (
                    <span className={styles.actividadSubtitulo}>
                      — {actividad.subtitulo}
                    </span>
                  )}
                </div>
              </div>
            </li>
          ))}
        </ul>
      </div>
    </div>
  </div>
)}
```

### 4. Body Scroll Lock

Prevents background scrolling when modal is open:

```typescript
useEffect(() => {
  if (selectedDay) {
    document.body.style.overflow = "hidden";
  } else {
    document.body.style.overflow = "unset";
  }
  return () => {
    document.body.style.overflow = "unset";
  };
}, [selectedDay]);
```
