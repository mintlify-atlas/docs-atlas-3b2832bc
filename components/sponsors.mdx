# Sponsors Component

The Sponsors component showcases the event organizers and celebrates the 50th anniversary of IESP Túpac Amaru. It features institutional logos, program information, and decorative elements.

## Overview

The Sponsors component provides:
- **Anniversary celebration** - 50th anniversary branding and messaging
- **Organizer logos** - Display of institutional and program logos
- **Program information** - Details about the Desarrollo de Sistemas de Información program
- **Sparkle animations** - Decorative animated elements

## Props

The Sponsors component does not accept any props. It's a self-contained section component.

## File Location

```

```

## Component Structure

The component consists of three main sections:

1. **Anniversary Section** - Celebrates the institution's 50-year history
2. **Organized By Section** - Shows organizer logos with links
3. **Career Info Section** - Details about the DSI program

```tsx

  <div className={styles.container}>

    <div className={styles.organizedSection}>...</div>
    
    {/* 3. Career information */}
    <div className={styles.careerInfo}>...</div>
  </div>
</section>
```

### 4. Fade-in Animation

Section animates into view on mount:

```typescript

}, []);
```

## Design Notes

### Two-Column Layout

The anniversary section uses a two-column layout:
- **Left**: Logo, tagline, website link, sparkles
- **Right**: Student illustration

This creates visual balance and draws attention to both the branding and the human element.

### Sparkle Decorations

The four sparkle elements are positioned around the logo using CSS absolute positioning. They add a celebratory feel appropriate for the 50th anniversary theme.

## Accessibility Features

- **Alt text**: All images have descriptive alt text
- **Semantic HTML**: Proper heading hierarchy (h3)
- **Link labels**: Links have descriptive text and context
- **External link handling**: `rel="noopener noreferrer"` prevents security issues
- **Target blank**: External links open in new tabs for better UX

## Customization

### Updating Anniversary Year

To update for future anniversaries, change the logo image and update the text in the JSX.

### Adding More Organizers

Add additional logo blocks to the `.organizedLogos` container:

```tsx
<a
  href="https://new-organizer.com"
  target="_blank"
  rel="noopener noreferrer"
  className={styles.organizerLogo}
>
  <img
    src="/new-organizer-logo.png"
    alt="New Organizer Name"
    className={styles.logoImage}
  />
</a>
```

### Removing Sparkles

To simplify the design, remove the sparkle span elements:

```tsx
<div className={styles.logoSection}>

```
