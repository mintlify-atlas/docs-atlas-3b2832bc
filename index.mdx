  <Steps>
    <Step title="Clone the repository">
      Clone the project from GitHub to your local machine:
      
      ```bash
      git clone https://github.com/leonardo007-code/semanainformatica2025.git
      cd semanainformatica2025
      ```
    </Step>

    <Step title="Install dependencies">
      Install all required packages using npm:
      
      ```bash
      npm install
      ```
      
      The project uses React 18, TypeScript, Vite, Tailwind CSS, anime.js, and other modern frontend tools.
    </Step>

    <Step title="Start the development server">
      Launch the local development server:
      
      ```bash
      npm run dev
      ```
      
      Open your browser and navigate to `http://localhost:5173` to see the application running.
    </Step>

    <Step title="Customize the content">
      Edit the event data in `src/data/schedule.ts` and `src/components/Speakers.tsx` to customize speakers, schedule, and event details for your needs.
    </Step>
  </Steps>
