## What is DEV WEEK 2025?

DEV WEEK is an annual student technology event organized by the Information Systems Development program at Instituto de Educación Superior Público Túpac Amaru (ISTTA) in Cusco, Peru. The 2025 edition takes place from December 10-12, 2025, as part of ISTTA's 50th anniversary celebration.

This single-page event website showcases the schedule, speakers, live streams, and registration system for the three-day technology conference featuring cybersecurity bootcamps, IT management talks, and student-led technical sessions.

## Key features

The DEV WEEK 2025 website includes:

- **Responsive hero section** with animated binary rain effect and countdown clock
- **Interactive schedule** showing daily events and speaker sessions
- **Speaker profiles** with filtering by day and expandable details
- **Live stream integration** supporting YouTube and Twitch embeds
- **Registration form** with validation and localStorage persistence
- **Smooth animations** powered by anime.js with accessibility support
- **SEO optimization** including structured data and Open Graph tags
- **Lazy loading** for optimal performance on all devices

## Project structure

The source code follows a component-based architecture:

```
src/
├── components/           # React components
│   ├── Hero.tsx         # Main hero section with CTA
│   ├── CountdownClock.tsx  # Event countdown timer
│   ├── Schedule.tsx     # Daily event schedule
│   ├── Speakers.tsx     # Speaker profiles with filtering
│   ├── LiveStreams.tsx  # Live stream modal
│   ├── RegistrationForm.tsx  # Registration with validation
│   ├── Sponsors.tsx     # Sponsor logos
│   └── Footer.tsx       # Site footer
├── utils/
│   ├── animations.js    # anime.js animation functions
│   └── streamHelpers.ts # Stream utility functions
├── data/
│   └── schedule.ts      # Event schedule data
├── types/               # TypeScript type definitions
├── __tests__/           # Unit tests
├── App.tsx              # Main app component
└── main.tsx             # Application entry point
```
