## Open Graph (Facebook)

Open Graph tags enable rich social media previews when the site is shared on Facebook, LinkedIn, and other platforms.

### Implementation

```html
<!-- index.html:40-57 -->
<!-- Open Graph / Facebook -->
<meta property="og:type" content="website" />
<meta
  property="og:url"
  content="https://semanainformatica2025.expandeya.com/"
/>
<meta property="og:site_name" content="DEV WEEK 2025 - IESP Túpac Amaru" />
<meta property="og:title" content="DEV WEEK 2025 - Semana del Desarrollo" />
<meta
  property="og:description"
  content="Únete al evento tecnológico de Cusco. 10-12 de diciembre 2025. Bootcamp de Ciberseguridad OWASP y Gestión de TI."
/>
<meta
  property="og:image"
  content="https://semanainformatica2025.expandeya.com/og-image.jpg"
/>
<meta property="og:image:width" content="1200" />
<meta property="og:image:height" content="630" />
```

### Open Graph Properties

| Property | Value | Purpose |
|----------|-------|----------|
| `og:type` | `website` | Defines content type |
| `og:url` | Full URL | Canonical URL for sharing |
| `og:site_name` | DEV WEEK 2025 - IESP Túpac Amaru | Site identifier |
| `og:title` | DEV WEEK 2025 - Semana del Desarrollo | Share title |
| `og:description` | Event description | Share description |
| `og:image` | /og-image.jpg | Social preview image |
| `og:image:width` | 1200 | Image width in pixels |
| `og:image:height` | 630 | Image height in pixels |

### Image Requirements

**Recommended dimensions:**
- Facebook/LinkedIn: 1200 × 630 px
- Minimum: 600 × 315 px
- Aspect ratio: 1.91:1

**Image guidelines:**
- Format: JPG or PNG
- File size: < 8 MB
- No text overlay (or minimal)
- High contrast, clear imagery

## Robots.txt

Controls search engine crawler access.

### Implementation

```txt
# public/robots.txt
User-agent: *
Allow: /

# Disallow admin and private areas (if any)
Disallow: /admin/
Disallow: /private/

# Allow all images
Allow: /*.png$
Allow: /*.jpg$
Allow: /*.jpeg$
Allow: /*.gif$
Allow: /*.svg$
Allow: /*.webp$

# Sitemap location
Sitemap: https://semanainformatica2025.expandeya.com/sitemap.xml

# Crawl-delay for respectful crawling
Crawl-delay: 1
```

### Directives Explained

| Directive | Value | Purpose |
|-----------|-------|----------|
| `User-agent` | * | Applies to all crawlers |
| `Allow` | / | Allow all pages |
| `Disallow` | /admin/, /private/ | Block these paths |
| `Allow` | Image extensions | Explicitly allow images |
| `Sitemap` | URL | Point to sitemap.xml |
| `Crawl-delay` | 1 | Wait 1 second between requests |

## Sitemap.xml

XML sitemap helps search engines discover and index pages.

### Implementation

```xml
<!-- public/sitemap.xml -->
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9"
        xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
        xsi:schemaLocation="http://www.sitemaps.org/schemas/sitemap/0.9
        http://www.sitemaps.org/schemas/sitemap/0.9/sitemap.xsd">
  
  <!-- Main page -->
  <url>
    <loc>https://semanainformatica2025.expandeya.com/</loc>
    <lastmod>2025-11-27</lastmod>
    <changefreq>daily</changefreq>
    <priority>1.0</priority>
  </url>
  
  <!-- Update lastmod when content changes -->
  <!-- Add more URLs as site grows -->
  
</urlset>
```

### XML Tags Explained

| Tag | Value | Purpose |
|-----|-------|----------|
| `<loc>` | URL | Page location |
| `<lastmod>` | 2025-11-27 | Last modified date (YYYY-MM-DD) |
| `<changefreq>` | daily | Update frequency hint |
| `<priority>` | 1.0 | Relative importance (0.0-1.0) |

**Change frequency values:**
- `always` - Document changes with every access
- `hourly` - Hourly updates
- `daily` - Daily updates (current)
- `weekly` - Weekly updates
- `monthly` - Monthly updates
- `yearly` - Yearly updates
- `never` - Archived content

### Submitting Sitemap

**Google Search Console:**
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Select your property
3. Navigate to Sitemaps
4. Enter: `https://semanainformatica2025.expandeya.com/sitemap.xml`
5. Click Submit

**Bing Webmaster Tools:**
1. Go to [Bing Webmaster](https://www.bing.com/webmasters)
2. Add/select site
3. Navigate to Sitemaps
4. Submit sitemap URL

## Core Web Vitals

Google's page experience signals for ranking:

### Metrics

**LCP (Largest Contentful Paint):**
- Target: < 2.5 seconds
- Current optimization: Lazy loading, code splitting

**FID (First Input Delay):**
- Target: < 100 milliseconds
- Current optimization: Minimal JavaScript blocking

**CLS (Cumulative Layout Shift):**
- Target: < 0.1
- Current optimization: Reserved space for images, no layout shifts

### Implementation Strategies

**Code Splitting:**
```tsx
// src/App.tsx:5-10
const About = lazy(() => import("./components/About"));
const Schedule = lazy(() => import("./components/Schedule"));
// ... more lazy imports
```

**Suspense Boundaries:**
```tsx
// src/App.tsx:24-26
<Suspense fallback={<div style={{ minHeight: "200px" }} />}>
  <About />
</Suspense>
```
