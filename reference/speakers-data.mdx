## Overview

The speakers data for DEV WEEK 2025 is stored in `src/components/Speakers.tsx` and defined as the `SPEAKERS_DATA` constant. This array contains information about all speakers, their roles, topics, and schedule.

## Data Location

**File**: `src/components/Speakers.tsx`

**Constant**: `SPEAKERS_DATA`

## Type Definition

### Speaker Interface

```typescript
interface Speaker {
  id: number;
  name: string;
  role: string;
  bio: string;
  topic: string;
  day: string;
  time: string;
  image: string;
}
```

## Field Descriptions

| Field | Type | Description |
|-------|------|-------------|
| `id` | `number` | Unique identifier for the speaker |
| `name` | `string` | Full name of the speaker |
| `role` | `string` | Professional role or affiliation |
| `bio` | `string` | Biography and expertise description |
| `topic` | `string` | Subject or topic the speaker will present |
| `day` | `string` | Day of presentation (e.g., "Miércoles 10", "Jueves 11", "Viernes 12") |
| `time` | `string` | Time slot for the presentation (e.g., "12:10 - 12:50") |
| `image` | `string` | Path to speaker's profile image |

## Usage Example

### Accessing Speaker Data

```typescript
import { SPEAKERS_DATA, type Speaker } from './Speakers';

// Get all speakers
const allSpeakers: Speaker[] = SPEAKERS_DATA;

// Get speakers for a specific day
const wednesdaySpeakers = SPEAKERS_DATA.filter(
  speaker => speaker.day === "Miércoles 10"
);

// Get OWASP speakers
const owaspSpeakers = SPEAKERS_DATA.filter(
  speaker => speaker.role.includes("OWASP")
);

// Search speakers by name or topic
const searchSpeakers = (term: string) => {
  return SPEAKERS_DATA.filter(
    speaker =>
      speaker.name.toLowerCase().includes(term.toLowerCase()) ||
      speaker.topic.toLowerCase().includes(term.toLowerCase()) ||
      speaker.role.toLowerCase().includes(term.toLowerCase())
  );
};
```

### Filtering Logic (From Component)

```typescript
const [selectedDay, setSelectedDay] = useState("Todos");
const [searchTerm, setSearchTerm] = useState("");

const filteredSpeakers = SPEAKERS_DATA.filter((speaker) => {
  const matchesDay = selectedDay === "Todos" || speaker.day === selectedDay;
  const matchesSearch =
    speaker.name.toLowerCase().includes(searchTerm.toLowerCase()) ||
    speaker.topic.toLowerCase().includes(searchTerm.toLowerCase()) ||
    speaker.role.toLowerCase().includes(searchTerm.toLowerCase());
  return matchesDay && matchesSearch;
});
```
